# AgriEdge-OMS-Salesforce
Salesforce-based Order Management System for agriculture supply chain

# Project Overview
AgriEdge Or-Mange Ltd is a leading organization in the agriculture and food production sector. This project focuses on developing a Salesforce-based Order Management System (OMS) to automate order processing, improve inventory tracking, and enhance customer satisfaction.

The system manages products such as seeds, fertilizers, harvested crops, and processed food items, ensuring efficient supply chain operations.

# Objectives
- Automate order processing to reduce manual errors
- Provide real-time inventory tracking
- Improve supply chain visibility
- Enhance customer service experience
- Ensure data security and compliance
- Enable data-driven decision making

# Existing Challenges
- Manual order processing errors
- Lack of real-time inventory visibility
- Disconnected customer service channels
- Delays in order fulfillment
- Risk of stockouts and overstocking

# Proposed Solution
A Salesforce-based Order Management System that:
- Automates the order lifecycle
- Tracks inventory in real-time
- Integrates customer service channels
- Provides dashboards and reports
- Ensures secure and role-based access

# System Features

## Order Management

- Order status tracking
